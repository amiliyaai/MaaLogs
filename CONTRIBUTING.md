# 贡献指南

欢迎贡献代码！请遵循以下规范。

## Commit 规范

### 格式

```
<类型>: <描述>

[可选的正文]
```

### 类型

| 类型 | 说明 |
|------|------|
| `feat` | 新功能 |
| `fix` | 修复 bug |
| `docs` | 文档更新 |
| `style` | 代码格式（不影响功能）|
| `refactor` | 重构（不影响功能）|
| `perf` | 性能优化 |
| `test` | 测试相关 |
| `chore` | 构建过程或辅助工具变动 |

### 示例

```
feat: 添加 AI 分析功能

新增 AI 智能分析，支持多个服务商

- OpenAI
- Claude
- DeepSeek

Close #123
```

```
fix: 修复模型切换不生效问题

切换服务商后，模型列表未同步更新
```

## Pull Request 规范

### 标题格式

```
<类型>: <描述>
```

示例：
- `feat(ai): 添加 DeepSeek 模型支持`
- `fix(parsing): 修复日志解析错误`

### PR 描述模板

```markdown
## 概述
简要描述本次改动

## 改动内容
- 改动1
- 改动2

## 测试
- [ ] 本地测试通过

## 截图（如有）
```
